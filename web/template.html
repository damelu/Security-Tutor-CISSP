Resuming Each Day with Your AI Tutor
This web UI is one piece of a larger study system. Each night you should also run a full AI-guided session that includes lecture analysis, homework quizzing, wrong-answer deep dives, and bank updates. Here's how:
  1. Open your course folder — In Cowork, click the folder icon and select your CISSP Training 2026 folder (the one with AGENTS.md, STUDY-GUIDE.md, Student files, etc). In VS Code / Cursor, open that same folder as your workspace.
  2. Start a new session — Open a new chat. Copy the prompt from DAILY-PROMPT.md and replace [DAY] with today (e.g., "Tuesday"). The AI reads AGENTS.md first, validates your files, then walks you through 5 phases.
  3. The 5 nightly phases: Lecture Analysis (transcripts) → Homework Quiz (tonight's questions) → Wrong Answer Deep Dive (explanations + memory hooks) → Audio Generation (study audio) → Bank Update (WRONG-ANSWER-BANK.md).
  4. Sync with this web UI: Connect your study folder in the "Folder Sync" tab and wrong answers auto-save to your folder and WRONG-ANSWER-BANK.md. No manual copy-paste needed.
  5. For a quick review quiz without the full nightly flow, use the prompt from REVIEW-PROMPT.md instead — it generates a 20–30 question quiz weighted by your weakest domains.
The key files are AGENTS.md (system context the AI reads first), DAILY-PROMPT.md (nightly workflow), REVIEW-PROMPT.md (on-demand review), STUDY-GUIDE.md (schedule + homework), and WRONG-ANSWER-BANK.md (your running log of missed questions).

What is the Wrong Answer Bank?
The WRONG-ANSWER-BANK.md file is a Markdown file in your course repo that stores detailed records of every question you've gotten wrong — including the correct answer, why you got it wrong, and a memory hook to help you remember next time.
When to use this: If you (or your AI tutor) have been maintaining a WRONG-ANSWER-BANK.md file from study sessions outside this web UI, you can import those entries here so they appear in the "Re-Quiz Wrong" tab for drilling.
How it works:
  1. Open your WRONG-ANSWER-BANK.md file (from your repo or wherever you keep it).
  2. Select all the content and copy it.
  3. Paste it into the text area below and click "Import Questions."
  4. The parser looks for entries in this format:
### Q42 | Domain 1 | Security and Risk Management **Question:** What is the primary purpose of... **Correct Answer:** B) Risk mitigation **Concept:** Risk Management Frameworks **Why Wrong:** Confused risk avoidance with mitigation **Memory Hook:** Mitigation = reduce, Avoidance = eliminate
Imported entries appear in "Re-Quiz Wrong" and won't be duplicated if you import the same file again. Questions imported from the bank are tagged as source: bank so you can tell them apart from questions you missed in quizzes.
